# Student Management System

## Description
A console-based Student Management System built using Python and SQLite. 
The application allows users to manage student records including student ID, name, course, and marks. 
Data is stored in a SQLite database to perform basic CRUD operations.

## Features
- Add new student records
- View all student details
- Update student information
- Delete student records
- Persistent storage using SQLite database

## Language & Tools
- Python
- SQLite (SQL)
- DB Browser for SQLite (optional tool)

## Skills Learned
- Python and SQL integration
- CRUD operations using SQL
- Database creation and table management
- Writing SQL queries inside Python
- Real-world database handling logic
